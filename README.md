# Basic Port Scanner 🔍

This is a simple Python-based port scanner developed as part of my cybersecurity practice.

## Features
- Scans common ports (1-1024)
- Detects open ports on a target IP
- Uses Python socket programming

## How to Run
1. Run the script
2. Enter target IP address
3. View open ports
